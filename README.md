## A Bit More About Me

I push CPTS from Hack The Box to anyone asking how to start — it teaches you to *think*, not just pass an exam. CTFs are great for sharpening your skills and exposing you to creative scenarios, but don't mistake them for real pentesting. In real engagements, sometimes you find nothing. You have to be okay with that and keep pushing.

Having a development background before going full offensive security was one of the best things that happened to my career. It's hard to test what you don't understand.
